Aula de git/GitHub

Sequência para uso do Git

mkdir projeto -> cria diretório

cd projeto -> entra na pasta diretório criado

git init -> inicia projeto Git

git status -> verifica status projeto Git

git add . -> adiciona para o stage 2° etapa

git commit -m "mensagem aqui" -> Marca Ponto ono tempo com o modificação criada

git log -> Ver os pontos comitados no projeto

ls -a lista diretórios

…or create a new repository on the command line

echo "# projetoGit" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/DeividGit/projetoGit.git
git push -u origin main

…or push an existing repository from the command line


git remote add origin https://github.com/DeividGit/projetoGit.git
git branch -M main
git push -u origin main
